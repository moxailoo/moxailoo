### Marco — `moxailoo`

Toronto-based dev who spends most weekends inside an Ethereum block explorer.

I started writing Solidity in 2022 because I wanted to know how a constant-product AMM actually balanced its books, and never really stopped. These days I mostly build small terminal tools for things I keep wanting to look up — pool state, position health, gas-cost breakdowns — instead of clicking through the same dashboards over and over.

I don't ship audits and I don't run a fund. I do read a lot of contract source.

---

#### What I'm into

- **Concentrated liquidity** — V3 math, range strategies, the awkward gap between "what the whitepaper says" and "what actually happens in a trending market"
- **Onchain reads at scale** — multicall, log decoding, archive nodes, building tools that don't fall over when you ask for a million ticks
- **MEV from the spectator side** — I run a couple of bots, mostly to lose money while learning, but I find the public-mempool game more interesting as a research subject than a trade
